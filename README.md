# life_pattern_tracker

Flutter application for tracking app usage, insights, and productivity patterns.

## Quick Start

Install dependencies:

```powershell
flutter pub get
```

Run app:

```powershell
flutter run
```

## Docker Workflow

Run setup + analyze + tests in Docker:

```powershell
.\docker\run.ps1
```
